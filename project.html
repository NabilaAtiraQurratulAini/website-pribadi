My Interface

1. Login

Pada halaman login user akan melakukan pengecekan terhadap username dan password yang dimasukkan oleh user. Ketika user berhasil untuk melakukan login ke web maka web menampilkan home seperti gambar di bawah ini.

2. Detail Comment

Menampilkan list komentar yang pernah diberikan oleh setiap user pada suatu task. Setiap komentar terdiri dari informasi isi komentar, user yang memberikan komentar, dan tanggal atau jam komentar diberikan.

3. Detail Task

User dapat melihat detail data task dengan menggunakan button dan muncul detail data task. Detail data task berisi informasi mengenai task id, nama task, nama project, nama module, status task, due data, initial estimate, remaining estimate, actual duration, description dan komentar.

4. Detail Storage

Tempat penyimpanan media hal-hal yang memberikan rasa damai yang di isi sendiri oleh user, seperti foto dan video berharga, atau e-book, atau bahkan lagu dari pihak ketiga (YouTube, Soundcloud) yang dapat di hubungkan dengan layanan cloud user.

5. Detail Customer

Sistem keanggotaan atau pasien premium yang dapat di akses dengan membayar secara bulanan untuk mendapatkan lebih banyak fitur seperti dapat konsultasi 24 jam, penyimpanan tambahan untuk galeri custom dan lebih banyak materi edukasi yang dapat di akses.
